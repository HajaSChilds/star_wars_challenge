## Star Wars Challenge
> Consumes the Official [Star Wars Api](https://swapi.dev/): **SWAPI** in a fun React Js project

This project uses:
- React Js
- Material UI - Timeline
- React Hooks 
- Fetch API 
- SCSS styling & animation
